EC2 Fundamentals Quiz
1. Which EC2 Purchasing Optio0n can provide you with the biggest discount, but it is not suitable for critical jobs or databases?
- Spot Instances

2. What should you use to control traffic in and out of EC2 Instances?
- Security Groups

3. How long can you reserve an EC2 Reserved Instance?
- 1 or 3 years

4. You would like to deploy a High-Performance Computing (HPC) application on EC2 Instances. Which EC2 Instance type should you choose?
- Compute Optimized

5. Which EC2 Purchasing Option should you use for an application you plan to run on a server continuously for 1 year?
- Reserved Instances

6. 
- 1. Preparing to launch an application that will be hosted on a set of EC2 Instances.
- 2. Software installation and some OS packages need to be updated during the first launch
- 3. Best way to achieve this when you launch the EC2 Instances?

= 1. Write a bash script that installs the required software and updates your OS
= 2. Use this script in EC2 User Data when you launch your EC2 Instances

7. Which EC2 Instance Type should you choose for a critical application that uses an in-memory database?
- Memory Optimized

8. 
- 1. E-commerce application with an OLTP Database hosted on-premises.
- 2. Thousands of requests per second. Migrate the database to an EC2 Instance
- 3. Which EC2 Instance Type should you choose to handle this high-frequency OLTP DB?

= Storage Optimized

9. Security Groups can be attached to only one EC2 Instance
- False

10.
- 1. Migrate on-premises applications to AWS.
- 2. Strict compliance requirements that require your applications to run on dedicated servers
- 3. Own server-bound software license to reduce costs.
- 4. Which EC2 Purchasing Option is suitable for you

= Dedicated Hosts
