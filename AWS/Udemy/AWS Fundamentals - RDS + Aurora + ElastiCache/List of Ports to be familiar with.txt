List of Ports to be familiar with
- 1. Important Ports (HTTPS - port 443)
  = 1. FTP: 21
  = 2. SSH: 22
  = 3. SFTP: 22 (same as SSH)
  = 4. HTTP: 80
  = 5: HTTPS: 443

- 2. DB Ports (PostgreSQL - port 5432)
  = 1. PostgreSQL: 5432
  = 2. MySQL: 3306
  = 3. Oracle RDS: 1521
  = 4. MSSQL Server: 1433
  = 5. MariaDB: 3306 (same as MySQL)
  = 6. Aurora: 5432 (if PostgreSQL compatible) or 3306 (if MySQL compatible)
